Zadání

Vytvořte webovou prezenatci nějakého dílčího úkolu z vaší diplomové práce.

Požadavky

Prezentace bude:

  • takového rozsahu, aby o ní bylo možno cca 5-10 minut mluvit,
  • bude kombinovat text, kód a grafiku
  • bude začleněná do tohoto github repozitáře

Postup

  1. Nainstalujte git
  2. Vytvořte si účet na https://github.com/
  3. Forkněte (tlačítko Fork) repo https://github.com/hanel/PED_web do svého repozitáře
  4. Naklonujte své repo (tlačítko Clone or download) - zkopírujte adresu *.git
  5. V Rstudiu založte nový projekt z Version Control a adresu *.git vložte
  6. V Rstudiu pomocí tlačítka Pull repozitář stáhněte

  7. Pomocí ctrl+shift+b sestavte web (nebo záložka Build)

  8. V RStudiu vytvořte nový RMarkdown dokument
  9. Pojmenujte ho např. vaším monogramem

  10. Hlavičku nechte ve tvaru (název upravte)

---
title: "Váš název"
---
  1. Do souboru napiště nějaký provizorní text
  2. Editací souboru _site.yml začleňte vaši prezentaci do menu, tj. v části hlavičky
 - text: Prezentace
    menu:
    - text: Jan Petr
      href: jp.html

doplňte další odrážku -text: Vaše jméno a href: odkaz_na_váš_soubor.html (nezapomeňte, že přestože editujete soubor odkaz_na_váš_soubor.Rmd, zde se objeví již přeložený soubor odkaz_na_váš_soubor.html)

  1. Projekt sestavte, v záložce Git zaregistrujte změny (Commit) - nezapomeňte přidat zprávu o tom, co jste udělali.
  2. Pomocí tlačitka Push projekt nahrajte na github

  3. Na githubu vytvořte nový pull request

  4. Kroky 12-13 průběžně opakujte, pokud budete chtít pracovat s repozitářem na jiném počítači, můžete jej synchronizovat pomocí tlačítka pull

  5. Dodělejte svou stránku

  6. Na hlavní stránce (index.Rmd) doplňte Vaše jméno, nějaký reprezentativní obrázek a abstrakt k vaší práci. Pozn.: obrázek by měl mít šířku 200px - můžete buď upravit v grafickém editoru nebo obrázek vložit pomocí <img src="img/VAS_OBRAZEK" style="width:200px"/>

  7. Pushněte na github a vytvořte pull request

  • Pokud chcete synchronizovat svůj repozitář s původním repozitářem (https://github.com/hanel/PED_web), do kterého možná již přispěli vaši spolužáci, zkuste použít následující skript
library(git2r)
repo = repository('cesta_k_vasemu_repozitari_v_pocitaci')
if (!"upstream" %in% remotes(repo)) remote_add(repo, 'upstream', 'https://github.com/hanel/PED_web.git')
fetch(repo, 'upstream')
merge('master', 'upstream')
